# ML Algorithm Visualizer

This project provides visualizations for various classical machine learning algorithms to illustrate how they work. The animations are created using Python, `matplotlib`, and `scikit-learn`.

## Visualizations

* Linear Regression
* K-Means Clustering
* Gradient Descent

## Usage

To run the visualizations, clone the repository, install the dependencies, and run the main script:

```bash
git clone <repository-url>
cd ml-visualizer
pip install -r requirements.txt
python main.py
```
